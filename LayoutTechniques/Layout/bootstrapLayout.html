Bootstrap Layout

3 column layout

The .row Class

The .row class makes the element display: flex, flex-wrap: wrap, and gives its children padding-left and right and margin-top

The .col-md-4 Class

.col-md-4 creates three equal-width columns above 768px giving a width: 33.3333%

The .container class

.container has five break points assigning a max-width to each. Thea base .container styles give it centering with margin: 0 auto and padding left and right.
